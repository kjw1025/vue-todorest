# Todo 앱 (kakao API)

## 프로젝트
db.json 을 이용해 목업서버 구축후 기본적인 CRUD 작업 <br>
kakao API 이용해 로그인 후 이용 가능하도록 설정

#### 사용기술 : Vue, JavaScript, CSS

##### 라이브러리 : axios, Vuex, vue-router, lodash, Bootstrap
